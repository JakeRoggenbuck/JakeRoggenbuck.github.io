# hexviewer-rs
Hexviewer allows you to quickly view the hex representation of a file with vim style keybinds<br><br>

# Install
```sh
git clone https://github.com/JakeRoggenbuck/hexviewer-rs
cargo install --path hexviewer-rs
```
