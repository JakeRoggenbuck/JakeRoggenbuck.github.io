# xdotool_python_window_util

## Use

### Size
```py
from main import Size

a = Size(40, 60)
a.size_win()
```

### Move
```py
from main import Move

a = Move(True, 20, 30)
a.move_win()
```

PYTHONPATH=./src pytest
