# vimage.vim
vimage.vim - Open image paths in vim

## Install
`Plug jakeroggenbuck/vimage.vim`

## Use
Set image viewer `let g:image_viewer = "feh"`<br>
Set key binding `nmap <leader>mg :call Vimage()<CR>`
