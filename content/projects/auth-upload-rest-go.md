# auth-upload-rest-go
Authenticated REST server written in Go.

## Usage
Upload files to a dedicated server for file sharing, etc.

## Running
```sh
# Run the server
cd backend
go run .
```

```sh
# Open the index.html with firefox - you can use whatever browser you like
firefox index.html
```
